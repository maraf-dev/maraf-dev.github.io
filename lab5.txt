Question 1: What are some of the benefits of using a static site generator? What are some disadvantages?
Advantages: It is easier and faster to set up and maintain becuase it has no server side code. It is also Secure against server side, or database attacks. 
Disadvantages: Does not allow for user interactions. real-time data updates are not supported.
